[English] This is a simple and lightweight command-line (CLI) stopwatch application built with Python.
It utilizes Python's built-in `time` module to measure elapsed time with millisecond-level precision 
between keypresses (Enter).



 Features

* Ճշգրիտ չափում (Precise Measurement): Օգտագործում է համակարգային ժամանակը (`time.time()`)՝ վայրկյանները
  2 տասնորդական նիշի ճշտությամբ չափելու համար:
* Պարզ կառավարում (Simple Controls):Կառավարվում է ընդամենը մեկ ստեղնով (Enter)՝ սկսելու և կանգնեցնելու համար:
* Ինտերակտիվ մենյու (Interactive Menu):Թույլ է տալիս կատարել բազմակի չափումներ կամ դուրս գալ ծրագրից ցանկացած
